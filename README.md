**React 打卡第八天**

1. **组件私有方法都用 `_` 开头**    (通用)
2. **事件监听方法都用 `handle` 开头**    (通用)
3. **事件监听方法传给组件的时候, 用 `on` 开头**    (通用)

**组件内容编写顺序 (参考): **

1. `static` 开头的类属性, 如 `defaultProps`, `propTypes`
2. 构造函数,  `constructor`
3. getter / setter
4. 组件生命周期
5. `_` 开头私有方法
6. `handle*` 事件监听方法
7.  `render*`开头的方法，有时候 `render()` 方法里面的内容会分开到不同函数里面进行，这些函数都以 `render*` 开头。
8. `render()` 方法
